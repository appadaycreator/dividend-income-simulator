FEATURES

配当収入シミュレーターの特徴

投資初心者からFIREを目指す方まで。配当のリアルな数字を、わかりやすく見える化します。

年間・月間の配当を自動計算

保有資産額と配当利回りを入力するだけで、税引き前後の年間・月間の配当収入を瞬時に算出します。

FIRE達成度を見える化

目標月額に対する達成率をプログレスバーとグラフで表示。あと何万円必要かまで具体的にわかります。

NISA/一般口座の税制差分

NISA（非課税）と一般口座（約20%課税）の手取り差をシミュレーション。賢い口座選びを後押しします。

HOW TO USE

FAQ

よくある質問

配当収入シミュレーターは無料で使えますか？
はい、完全無料・登録不要でご利用いただけます。回数制限もありません。
入力したデータはサーバーに送信されますか？
いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。
NISAと一般口座の違いも計算できますか？
はい。税引き後利率を切り替えることで、NISA（非課税）と一般口座（約20%課税）の手取り差を比較できます。
スマートフォンでも使えますか？
はい、スマートフォン・タブレット・PCすべてに最適化されています。
FIRE達成度はどう計算されますか？
入力した目標月額に対する手取り月間配当の割合を達成率として表示し、不足資産額も自動算出します。
